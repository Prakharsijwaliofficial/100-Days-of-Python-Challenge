# 🏓 Pong Game - Python Turtle Graphics

A classic **Pong game** built using Python's Turtle Graphics module.  
This project is part of my journey learning **Object-Oriented Programming (OOP)**, game loops, collision detection, and basic game physics.

## 🎮 Features

- Two-player Pong gameplay
- Player-controlled paddles
- Moving ball with realistic bouncing
- Collision detection with:
  - Top and bottom walls
  - Both paddles
- Score tracking system
- Smooth animation using Turtle graphics
- Object-Oriented Programming structure

---

## 🛠️ Technologies Used

- Python 3
- Turtle Graphics Library
- Object-Oriented Programming (OOP)
