# 🐢 Turtle Crossy Road Game

A fun **Crossy Road inspired game** built using Python's Turtle graphics module.

This project is part of my journey through **Angela Yu's 100 Days of Code: Python Bootcamp (Day 23)**, where I learned about:
- Object-Oriented Programming (OOP)
- Classes and Objects
- Game loops
- Collision detection
- Managing multiple objects
- Turtle graphics

---

## 🎮 Game Description

Help the turtle cross the road while avoiding randomly generated cars.

Every time the turtle successfully reaches the other side:
- The score increases
- The difficulty increases
- Cars move faster

The goal is to survive as long as possible and achieve the highest score.

---

## 🚀 Features

✅ Player-controlled turtle  
✅ Randomly generated cars  
✅ Different colored cars  
✅ Collision detection  
✅ Score tracking system  
✅ Level progression  
✅ Increasing difficulty  
✅ Game Over screen

## 🧠 Concepts Learned

### Object-Oriented Programming

The game is divided into different classes:

### 🐢 Player Class
Handles:
- Turtle movement
- Player position
- Reaching the finish line

### 🚗 CarManager Class
Handles:
- Creating cars
- Storing cars in a list
- Moving all cars
- Increasing car speed

### 🏆 Scoreboard Class
Handles:
- Score display
- Level updates
- Game over message
