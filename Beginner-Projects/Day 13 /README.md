# Day 13 - Coding Challenges

This repository contains the project for Day 13 of the 100 Days of Code Challenge. It features a centralized, interactive command-line dashboard that allows you to run three classic programming exercises.

## 📌 Features

* **Odd or Even Checker**: Quickly determines whether a given integer is odd or even using the modulo operator.
* **Leap Year Validator**: Evaluates a year through nested logic parameters to verify if it qualifies as a leap year.
* **FizzBuzz Game**: Counts from 1 up to your target number, replacing multiples of 3 with "Fizz", multiples of 5 with "Buzz", and multiples of both with "FizzBuzz".

## 🚀 How to Run

1. Make sure you have Python installed on your system.
2. Download or clone this directory.
3. Open your terminal or command prompt and navigate to the project folder.
4. Execute the script by running:

```bash
python main.py
```

## 🛠️ Usage Example

When you run the script, you will see a command-line interface menu:

```text
--- Day 13 Challenge Dashboard ---
1. Run Odd or Even Checker
2. Run Leap Year Validator
3. Run FizzBuzz

Choose a challenge to test (1-3): 
```

Simply type `1`, `2`, or `3` and follow the on-screen prompts to input your numbers and view the results.
