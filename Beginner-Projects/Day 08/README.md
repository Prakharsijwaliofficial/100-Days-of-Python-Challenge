# Day 8 - Caesar Cipher

### 📌 Project
A secure communication program that encrypts and decrypts messages by shifting letters along the alphabet based on a user-defined "shift" number.

### 🧠 What I Learned
*   Functions with Inputs
*   Modulo Operator (%)
*   List Indexing and Manipulation
*   While Loops and Logic Gates
*   Handling Symbols and Spaces

### 🚀 How to Run
1. Launch the program in your terminal.
2. Enter 'encode' or 'decode'.
3. Input your message and shift number.
