Day 7 - Hangman Game
📌 Project
A classic word-guessing game where the player tries to figure out a secret word letter-by-letter before they run out of lives and the "Hangman" is fully drawn.
🧠 What I Learned
Lists & Strings: Converting strings to lists and manipulating them via indices.
Game Loops: Using while loops to keep the game running until a win/loss condition.
Enumeration: Using enumerate() to find and update multiple letter positions at once.
Control Flow: Implementing if/elif/else logic to handle correct/incorrect guesses and duplicate inputs.
ASCII Art: Integrating multi-line strings to provide visual game states.
🚀 How to Run
Run the script: python hangman.py
Guess one letter at a time when prompted.
Try to complete the word before the 6 lives are gone!
