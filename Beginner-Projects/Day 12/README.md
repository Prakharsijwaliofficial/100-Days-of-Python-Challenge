# Day 12 - Number Guessing Game

## 📌 Project
A Python-based Number Guessing Game where the player competes against the computer to guess a secret number between 1 and 100 within a limited number of attempts based on the chosen difficulty.

## 💬 What I Learned
* Global vs. Local Scope
* Python Constants and Block Scope
* Loop control flows (`break` statements)
* Off-by-one error handling in loops
* User input validation and formatting

## 🚀 How to Run
* Run the code in any Python environment (like PyCharm, VS Code, or Replit).
* Choose a difficulty level by typing `easy` or `hard`.
* Input your guesses and follow the hints (`Too high!!` or `Too low!!`) until you win or run out of turns.
