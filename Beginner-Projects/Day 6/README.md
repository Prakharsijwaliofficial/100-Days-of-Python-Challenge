Day 6 - Hurdle Navigation Bot

📌 Project
A Python program that helps a robot navigate through obstacles (hurdles) and reach the goal using logical decision-making.

🧠 What I Learned

Functions
Conditional statements (if/elif/else)
While loops
Code reusability
Problem-solving logic

🚀 How to Run

Run the code in a platform like Reeborg's World
Click “Run” to watch the robot navigate obstacles
The robot will automatically find its way to the goal

💡 How It Works

If the right path is clear → turn right and move
If the front is clear → move forward
If blocked → turn left to find a new path
