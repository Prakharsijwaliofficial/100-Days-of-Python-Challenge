# Day 9 - Secret Auction

## 📌 Project

A silent auction program that collects bids from multiple users and determines the highest bidder without revealing other bids.

## 🧠 What I Learned

* **Python Dictionaries:** How to store data in key-value pairs.
* **Nesting:** Managing lists inside dictionaries and dictionaries inside lists.
* **Clearing the Console:** Using logic to clear the screen for the next user.

## 💻 How to Use

1. Run the `main.py` file.
2. Enter your name and your bid amount.
3. If there are other bidders, type 'yes' to clear the screen for the next person.
4. Once all bids are in, type 'no' to find out who won!
